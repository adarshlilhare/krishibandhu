# KrishiBandhu - AI Crop Assistant

KrishiBandhu is an AI-powered platform designed to assist farmers with disease detection, crop advisory, and market insights.

## 🌐 Live Demo

**[https://krishnabandhu.vercel.app](https://krishnabandhu.vercel.app)**

## Features

- **Disease Detection** — Upload plant leaf images to detect diseases using AI vision.
- **Crop Advisory** — Get personalized recommendations based on soil and climate parameters.
- **Market Insights** — View real-time market prices from local Mandis.
- **Voice Navigation** — Navigate the app hands-free using voice commands.
- **Mobile App** — Flutter-based Android wrapper available in the `mobile` directory.
